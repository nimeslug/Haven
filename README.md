# Haven 🏡

Masaüstü pet uygulaması. Şu an desteklenen: Pamuk 🐰

## Kurulum

```powershell
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python src\main.py
```

## Özellikler

- 🐇 Şeffaf, çerçevesiz, sürüklenebilir masaüstü peti
- 💤 Uyku modu (uzun süre etkileşimsiz kalırsa)
- 🎯 Ekranda zıplayarak hareket
- 💬 Emoji baloncukları
- 👆 Tıklama tepkisi (mutluluk zıplaması)
- 🔄 Sağ tık menüsü ile kontrol
- 🐹 Çoklu pet desteği (yeni hayvan eklemek için `assets/` altına klasör aç)

## Teknoloji

Python 3.12 + PySide6 (Qt)
